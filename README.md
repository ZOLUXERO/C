# para compilar un archivo .c
```
gcc <archivo>.c
```
# para ejecutar el archivo
```
./<archivo>.out
```

# se necesita instalar gdb para que funcione el debugger correctamente
```
sudo apt-get update
sudo apt-get install gdb
```
para saber la ruta del gdb
```
which gdb
```
